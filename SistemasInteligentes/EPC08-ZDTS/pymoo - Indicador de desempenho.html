Indicador de desempenho ¶

É fundamental para qualquer algoritmo medir o desempenho. Em um cenário multi-objetivo, não podemos calcular a distância para o ótimo global verdadeiro, mas devemos considerar um conjunto de soluções. Além disso, às vezes o ótimo nem mesmo é conhecido, e outras técnicas devem ser utilizadas.

Primeiro, vamos considerar um cenário em que a frente de Pareto é conhecida:

[1]:

import numpy as np
from pymoo.factory import get_problem
from pymoo.visualization.scatter import Scatter

# The pareto front of a scaled zdt1 problem
pf = get_problem("zdt1").pareto_front()

# The result found by an algorithm
A = pf[::10] * 1.1

# plot the result
Scatter(legend=True).add(pf, label="Pareto-front").add(A, label="Result").show()

Distância Geracional (GD) ¶

O indicador de desempenho GD [22] mede a distância da solução até a frente de Pareto. Vamos supor que os pontos encontrados pelo nosso algoritmo são o conjunto de vetores objetivoA={a1,a2,…,a|A|}A={a1,a2,…,a|A|} e o conjunto de pontos de referência (frente de Pareto) é Z={z1,z2,…,z|Z|}Z={z1,z2,…,z|Z|}. Então,

GD ( A )=1| A |(∑i = 1| A |dpEu)1 / pGD(UMA)=1|UMA|(∑Eu=1|UMA|dEup)1/p

Onde didi representa a distância euclidiana (p = 2) de aiai para seu ponto de referência mais próximo em ZZ. Basicamente, isso resulta na distância média de qualquer pontoAA para o ponto mais próximo na frente de Pareto.

[2]:

from pymoo.factory import get_performance_indicator

gd = get_performance_indicator("gd", pf)
print("GD", gd.calc(A))

GD 0,05497689467314528

Distância Geracional Plus (GD +) ¶

Ishibushi et. al. proposto em [23] GD +:

GD+( A )=1| A |(∑i = 1| A |d+Eu2)1 / 2GD+(UMA)=1|UMA|(∑Eu=1|UMA|dEu+2)1/2

onde para minimização d+i=max{ai−zi,0}di+=max{ai−zi,0} representa a distância modificada de aiai para seu ponto de referência mais próximo em ZZ com o valor correspondente zizi.

[3]:

from pymoo.factory import get_performance_indicator

gd_plus = get_performance_indicator("gd+", pf)
print("GD+", gd_plus.calc(A))

GD + 0,05497689467314528

Distância Geracional Invertida (IGD) ¶

O indicador de desempenho IGD [24] inverte a distância geracional e mede a distância de qualquer ponto emZZ para o ponto mais próximo em AA.

IGD ( A )=1| Z|(∑i = 1| Z|dEu^p)1 / pIGD(UMA)=1|Z|(∑Eu=1|Z|dEu^p)1/p

Onde di^di^ representa a distância euclidiana (p = 2) de zizi para seu ponto de referência mais próximo em AA.

[4]:

from pymoo.factory import get_performance_indicator

igd = get_performance_indicator("igd", pf)
print("IGD", igd.calc(A))

IGD 0,06690908300327662

Distância Geracional Invertida Plus (IGD +) ¶

Em [23] Ishibushi et. al. IGD + proposto, que é fracamente compatível com Pareto, enquanto o IGD original não é.

IGD+( A )=1| Z|(∑i = 1| Z|d+Eu2)1 / 2IGD+(UMA)=1|Z|(∑Eu=1|Z|dEu+2)1/2

onde para minimização d+i=max{ai−zi,0}di+=max{ai−zi,0} representa a distância modificada de zizi para a solução mais próxima em AA com o valor correspondente aiai.

[5]:

from pymoo.factory import get_performance_indicator

igd_plus = get_performance_indicator("igd+", pf)
print("IGD+", igd_plus.calc(A))

IGD + 0,06466828842775944

Hipervolume ¶

Para todos os indicadores de desempenho apresentados até agora, um conjunto de metas precisa ser conhecido. Para o hipervolume, apenas um ponto de referência precisa ser fornecido. Primeiramente, gostaria de mencionar que estamos usando a implementação Hypervolume do DEAP . Ele calcula a área / volume, que é dominado pelo conjunto de soluções fornecidas em relação a um ponto de referência.

Esta imagem foi tirada de [25] e ilustra um exemplo de dois objetivos onde a área que é dominada por um conjunto de pontos é mostrada em cinza. Considerando que para as outras métricas, o objetivo era minimizar a distância para a frente de Pareto, aqui, desejamos maximizar a métrica de desempenho.

[6]:

from pymoo.factory import get_performance_indicator

hv = get_performance_indicator("hv", ref_point=np.array([1.2, 1.2]))
print("hv", hv.calc(A))

hv 0.9631646448182305
